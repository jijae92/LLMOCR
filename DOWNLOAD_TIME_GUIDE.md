## ⚡ 왜 오래 걸리나요?

### 1. Hugging Face 서버 속도
- 전 세계 사용자가 동시 접속
- 서버 부하에 따라 속도 변동
- 시간대별로 속도 차이 발생

### 2. 이미지 파일 크기
- 각 이미지: 평균 **100KB**
- 고해상도 문서 이미지
- 압축되지 않은 원본 품질

### 3. 처리 과정
```
다운로드 → 저장 → 검증 → 메타데이터 생성
```
단순 다운로드뿐만 아니라 파일 처리 시간 포함

```

### 3️⃣ 최적 시간대 이용
```

📊 주말: 평일보다 빠름
⚠️ 평일 오후: 가장 느림
```

### 4️⃣ 유선 랜 사용
```
WiFi: 20~50 Mbps
유선: 100~1000 Mbps
→ 2~5배 빠름
```

### 5️⃣ 백그라운드 실행
```

✅ 다른 탭 사용 가능
✅ 다른 작업 가능

## ⚠️ 주의사항

### 다운로드 중단 시
```
✅ Start Index를 조정하여 이어받기 가능
❌ 처음부터 다시 받을 필요 없음
```

### 디스크 공간 부족 시
```
Error: No space left on device

해결:
1. Sample Limit 줄이기
2. 불필요한 파일 삭제
3. 외장 하드 사용
```

### 네트워크 오류 시
```
Error: Connection timeout

해결:
1. 인터넷 연결 확인
2. 방화벽 설정 확인
3. VPN 비활성화
4. 시간대 변경 후 재시도
```
